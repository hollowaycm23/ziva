# Skill: Weather Manager

Fornece informações meteorológicas detalhadas, incluindo temperatura, ventos, precipitação e índices de qualidade do ar.

## Funcionalidades

- Previsão do tempo para até 16 dias.
- Geocodificação automática de cidades.
- Detecção de níveis de AQI (Air Quality Index).
