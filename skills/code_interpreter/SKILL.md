# Skill: Code Interpreter

Permite ao Ziva interagir com o sistema de arquivos para criar scripts e pesquisar padrões técnicos em sua base de conhecimento.

## Funcionalidades

- Gravação de arquivos (Python, JS, etc).
- Recuperação de snippets técnicos via RAG.
