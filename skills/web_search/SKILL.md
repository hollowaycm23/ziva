# Skill: Web Search

Esta skill fornece capacidades de pesquisa na internet para o Ziva, utilizando o SearxNG como fonte primária e fallbacks para Brave e DuckDuckGo.

## Funcionalidades

- Pesquisa multi-motor.
- Auto-indexação de resultados na memória de longo prazo.
- Fallback automático em caso de erro 403 ou indisponibilidade.
